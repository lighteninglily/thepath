HOW IT WORKS:
1. User types content: "1. Faith Requires Action"
2. System waits 800ms for typing to stop
3. Calls Claude API with content
4. Claude analyzes and returns:
   - Best template ID (point-numbered-bold)
   - Extracted placeholders (number: "1", title: "Faith Requires Action")
   - Confidence score (0.95)
   - Reasoning
5. System applies template with placeholders
6. Beautiful formatted slide appears

AI CAPABILITIES:
- Detects: Scripture references, numbered points, questions, quotes
- Extracts: Numbers (1,2,A,I), titles, body text, verse references, authors
- Selects: Best template from 10+ options (title, scripture, point, quote, etc.)
- Highlights: Key emphasis words
